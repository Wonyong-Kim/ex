[추가] 딸림음은 낭송음, 걸림(suspension) 혹은 긴장의 음이고 마침음은 정지의 음이다. 이런 면에서 이들의 기능은 화성법에서의 딸림음 및 으뜸음과는 다르다. 2. 딸림음의 위치 -다성선법에서 딸림음의 위치는 현대의 음계에서와 같이 선법의 딸림음이 마침음의 완전5도 위에 예외 없이 위치하지만 -그레고리 선법중 프리지아(프레지아-그레고리안 선법)에서는 딸림음의 위치가 완전5도 위의 음인 B음 대신 C음으로 대치된다. 이것은 B음은 변하기 쉬운 음이기 때문에 딸림음이란 중요한 기능을 하기에는 적합하지 않기 때문이다. 3. 정격선법과 변격선법 르네상스 이후의 다성적 선법에서는 정격선법과 정격선법의 구분없이 정격선법을 사용하게 된다. 따라서 정격선법만 제대로 알고 있으면 큰 무리는 없지만 변격선법이 무엇인지 정도는 알 필요가 있다. 이 두 선법은 음계의 음역에 관계해서 마침음의 위치에 따라 결정되는데, 1) 정격선법 선율이 마침음과 마침음 사이에 있을 경우 정격선법이라고 하고, 2) 변격선법 선율의 음역이 마침음의 4도 아래부터 5도위까지일 경우를 변격선법이라고 한다. 4. 장3화음의 사용 르네상스의 다성 선법에서는 딸림음이나 마침음 위에 장3화음을 즐겨사용했다. [이끔음] 딸림음 위에 장3화음은 이끔음이 되는데 반드시 마침에서만 사용하였다. 각 이끔음은 아래와 같다. F# : 밐소리디아 C# : 도리아 G# : 에이올리아 F : 프리지아(프리지아는 D#을 사용하지 않고 upper leading tone인 F를 사용하였다.) [피카디3도] 마침음 위에 장3화음은 피카디3도(Tierce de Picardie)라고 부른다. 이로 인해 사용되는 임시표는 다음과 같다. F# : 도리아 C# : 에이올리아 G# : 프리지아 5. 선법의 선율 선법의 선율은 어떤 음에서도 가능하나 마침은 반드시 마침음에서 이루어져야 한다.
